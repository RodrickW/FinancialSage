# App Assets

This folder needs the following image assets for Expo:

1. **icon.png** - App icon (1024x1024 px)
2. **splash.png** - Splash screen (1242x2436 px)
3. **adaptive-icon.png** - Android adaptive icon (1024x1024 px)
4. **favicon.png** - Web favicon (48x48 px)

## Temporary Workaround

For now, Expo will use default icons during development and building.
You can add your custom images later by placing them in this folder.

## Where to get images

- Use your existing web app logo from: `../attached_assets/`
- Or create new ones at: https://icon.kitchen/
